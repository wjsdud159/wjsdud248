Animation


Animation

animation : {name} {durationn} {timing-function} {dealy} {iteration-count} {direction} {fill-mode} {paly-state}
property Description
animation-name @keyframes에 지정된 이름을 설정합니다.
animation-durationn 애니메이션이 실햄되는 시간
animation-timing-function 애니메이션 키프레임 움직임을 설정합니다.
animation-dealy 애니메이션이 모두 된 후 애니매이션이 다시 시작될 때 까지의 시간을 설정합니다.
animation-iterationcount 애니메이션 반복 횟수를 설정합니다.
animation-direction 애니메이션 움직임 방향을 설정해 줍니다.
animation-fill-mode 애니메이션이 끝난 후의 상태를 설정합니다.
animation-paly-state 애니메이션 진행 상태를 설정합니다.

timing-function

property Description
linear 일정한 간격으로 움직입니다.
ease 처음에 서서히 가속하다가 마지막 부분에 급격히 감속합니다.
ease-in 처음에는 천천히 시작하고 마지막에 가속합니다.
ease-out 처음에는 최대 속도로 시작하고 마지막에 감속합니다.
ease-in-out 처음에는 제로 속도에서 시작하고 중간시점에서 최고 속도로 움직이고, 마지막 부분에 서서히 감속합니다.
stap-start 에니매이션이 시작점에서만 표현 됩니다.
stap-end 에니메이션이 끝점에서만 표현됩니다.
staps(ints, start/end) 애니메이션 단계별로 표현합니다.
cubic-bezier(n,n,n) 베이지 곡선값을 만들어서 속도를 설정합니다.

Transition

transition : {property} {durationn} {timing-function} {dealy}
property Description
transition-property 트랜지션을 적용할 CSS 속성을 대상을 설정합니다.
transition-durationn 트랜지션 작동시간을 설정합니다.
transition-timing-function 트랜지션 움직임 효과를 성정합니다.
transition-dealy 트랜지션이 시작되기 전에 대기시간을 설정합니다.
